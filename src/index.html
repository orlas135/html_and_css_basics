Etiquetas de bloque VS Etiquetas de línea

Una etiqueta de bloque es aquella que genera saltos de línea con sus etiquetas hermanas
Una etiqueta de línea es aquella que solo ocupa el espacio necesario que tiene su contenido.

La etiqueta de bloque por excelencia es DIV

Etiqueta DIV

La etiqueta de línea por excelencia es span.

Etiqueta span Soy otra etiqueta span

Formas de insertar código CSS en un documento HTML

  • En la misma etiqueta con el atributo style (no recomendado)

  • Con una hoja de estilos a parte e insetándola con la etiqueta link en la cabecera de la página

  • Con la etiqueta style (de preferencia en la cabecera)

Formas de insertar código JavaScript en un documento HTML

  • Con la etiqueta script al final del documento antes del cierre de la etiqueta body

  • Con una hoja de scripts a parte y utilizando la etiqueta scripts con el atributo src y poniendo la ruta donde se encuentra la hoja
